# ART-Badge 应用开发

![ART-Badge](figures/ART-Badge.png)

应用开发是 PersimWear OS（湃心 OS）最上层软件开发过程，目前开发方式采用 JS 小程序的模式。

底层基于 JS 应用程序框架，框架中提供了众多组件支持，保证了上层应用开发的基础能力。
